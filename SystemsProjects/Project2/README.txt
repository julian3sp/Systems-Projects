Description:
This program takes sales data from a coffee shop and all the items its sold over the past year.
It stores and organizes this data for the user and depending on the users input, 
it will either print an items sales for each month, print the overall sales for 
each item to a file (sales.txt), or return the best selling item according to the data.

Compile/Running Program Instructions: (typed into terminal)
Compile: Type 'make all' into the terminal
Running: './register'
and then enter a number 1-3 when prompted
